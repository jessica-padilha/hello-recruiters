# Tech Recruiter Terminal Interaction

Este é um pequeno script em Python que simula uma interação simples no terminal.  
Ele pergunta ao usuário se ele é um Tech Recruiter e exibe uma mensagem apropriada com base na resposta.

## Funcionalidades:
- Pergunta ao usuário se ele é um tech recruiter.
- Aceita as respostas "yes" ou "no".
- Exibe uma mensagem personalizada de acordo com a resposta.
- Inclui um link para meu perfil no LinkedIn.

## Como executar:
1. Certifique-se de ter o Python instalado.
2. Salve o arquivo como `oportunity.py` (ou o nome que quiser).
3. Abra o terminal na pasta onde o arquivo está salvo.
4. Execute.
